10. МАСЪАЛАҲО БО ИСТИФОДАИ ТАКРОРШАВИИ ШАРТӢ

 

10.1. Ададҳои мусбии A ва B (A> B) дода шудаанд. Дар порчаи дарозиаш А шумораи максималии порчаи дарозии В ҷойгир карда шудаанд (бидуни болои ҳамдигар хобидан). Бе истифодаи амалҳои зарб ва тақсим дарозии қисми холии порчаи А-ро ёбед.

10.2. Ададҳои мусбии A ва B (A> B) дода шудаанд. Дар порчаи дарозиаш А шумораи максималии порчаи дарозии В ҷойгир карда шудаанд (бидуни болои ҳамдигар хобидан). Бе истифодаи амалҳои зарб ва тақсим миқдори порчаҳои B-ро ёбед.

10.3. Ададҳои мусбии N ва K дода шудаанд. Танҳо бо истифодаи амалҳои ҷамъ ва тарҳ, миқдори тақсими бутуни N ба K ва бақияи боқимондаи натиҷаи тақсимро ёбед.

10.4. Адади бутуни N (N> 0) дода шудааст. Агар он дараҷаи адади 3 бошад, он гоҳ сатри “ҳа”, вагарна сатри “не”-ро чоп кунед.

10.5. Адади бутуни N (N> 0) дода шудааст, ки ба дараҷаи адади 2 баробар аст: N = 2K. Адади K - нишондиҳандаи ин дараҷаро ёбед.

10.6. Адади бутуни N (N> 0) дода шудааст. Қимати факториали дукаратаи N-ро ёбед.

N!! = N·(N−2)·(N−4)·…

10.7. Адади бутуни N (N> 0) дода шудааст. Хурдтарин адади мусбати К-ро ёбед, ки квадрати он аз адади N зиёд бошад (K2> N). Функсияи решаи квадратиро истифода набаред.

10.8. Адади бутуни N (N> 0) дода шудааст. Калонтарин адади K-ро ёбед, ки квадрати он аз N зиёд набошад (K2 ≤ N). Функсияи решаи квадратиро истифода набаред.

10.9. Адади бутуни N (N> 0) дода шудааст. Хурдтарин адади бутуни K-ро ёбед, ки нобаробарии 3K> N барои он иҷро шавад.

10.10. Адади бутуни N (N> 0) дода шудааст. Калонтарин адади K-ро ёбед, ки нобаробарии 3K <N барои он иҷро шавад.

10.11. Адади бутуни N (N> 0) дода шудааст. Хурдтарин адади K-ро чоп кунед, ки барои он ифодаи 1 + 2 + ... + K аз адади N зиёд бошад. Ҳосили ҷамъи ифодаро низ чоп кунед.

10.12. Адади бутуни N (N> 0) дода шудааст. Калонтарин адади K-ро чоп кунед, ки барои он ифодаи 1 + 2 + ... + K аз адади N хурд ва ё баробар бошад.

10.13. Адади бутуни A (A > 1) дода шудааст. Хурдтарин адади K-ро ёбед, ки барои он ҳосили ҷамъи ифодаи  1 + 1/2 + ... + 1/K аз адади A калон бошад.

10.14. Адади бутуни A (A > 1) дода шудааст. Калонтарин адади K-ро ёбед, ки барои он ҳосили ҷамъи ифодаи  1 + 1/2 + ... + 1/K аз адади A хурд ва ё баробар бошад.

10.15. Ба бонк 1000 сомонӣ гузошта шуд. Баъди ҳар моҳ маблағи дар бонк буда ба P фоиз зиёд мешавад (P адади ҳақиқӣ ва 0 < P < 25). Аз рӯи P-и додашуда муайян кунед, ки баъди чанд моҳ маблағ аз 1100 сомонӣ мегузарад. Миқдори моҳҳои гузашта ва қимати маблағи ҷамъшударо чоп кунед.

10.16. Варзишгар рӯзи якум 10 км давида, тайёриро ба мусобиқаҳо сар кард. Ҳар рӯз ӯ масофаи давиданро ба P фоиз зиёд кард. (P - адади ҳақиқӣ ва 0 < P < 50). Аз рӯи P-и додашуда муайян кунед, ки баъди чанд рӯз масофа аз 200 км мегузарад. Миқдори рӯзҳои гузашта K (бутун) ва масофаи рӯзи охирон S (адади ҳақиқӣ) чоп карда шавад.

10.17. Адади бутуни N (N > 0) дода шудааст. Бо истифода аз амалҳои тақсими бутун ва боқимондаи тақсим, ҳамаи рақамҳои ададро дар сатрҳои алоҳида чоп кунед.

10.18. Адади бутуни N (N > 0) дода шудааст. Бо истифода аз амалҳои тақсими бутун ва боқимондаи тақсим, миқдор ва ҳосили ҷамъи рақамҳои ададро ёбед.

10.19. Адади бутуни N (N > 0) дода шудааст. Бо истифода аз амалҳои тақсими бутун ва боқимондаи тақсим, ҳамаи рақамҳои ададро ба таври чаппа чоп кунед (мисол, адади 6587 ба 7856).

10.20. Адади бутуни N (N > 0) дода шудааст. Бо истифода аз амалҳои тақсими бутун ва боқимондаи тақсим, муайян кунед, ки дар таркиби адади N рақами 2 ҳаст ё не.

10.21. Адади бутуни N (N > 0) дода шудааст. Бо истифода аз амалҳои тақсими бутун ва боқимондаи тақсим, муайян кунед, ки дар таркиби адади N ақаллан якто рақами тоқ ҳаст ё не.

10.22. Адади бутуни N (N > 0) дода шудааст. Муайян кунед, ки ин адад содда ҳаст ё не.

10.23. Ададҳои бутуни мусбати A ва B (A > B) дода шудаанд. Калонтарин тақсимкунандаи умумӣ (КТУ)  бо истифодаи алгоритми Евклид ёфта шавад.

КТУ(A, B) = КТУ (B, A mod B), агар B ≠ 0;   КТУ (A, 0) = A.

10.24. Адади бутуни N (N > 1) дода шудааст. Пайдарпайии ададҳои Фибоначчи FK чунин муайян карда мешавад: F1 = 1, F2 = 1, FK = FK–2 + FK–1, k = 3, 4, … . Тафтиш кунед, ки адади N адади Фибоначчи аст ё не?

10.25. Адади бутуни N (N > 1) дода шудааст. Аввалин адади Фибоначиро, ки аз адади N калон аст, ёбед (ба мисоли 24 нигаред).

10.26. Адади бутуни N (N > 1) дода шудааст, адади Фибоначиро ташкил медиҳад (ба мисоли 24 нигаред). Ададҳои қаблӣ ва навбатии Фибоначиро барои адади N ёбед.

10.27. Адади ҳақиқии ε (ε > 0) дода шудааст. Пайдарпайии ададҳои воқеии AK чунин муайян карда мешавад:

A1 = 1, AK = 1 + 2 / AK-1, K = 2, 3, …

Аввалин адади К-ро ёбед, ки шарти |AK - AK-1| <ε иҷро шавад.

10.28. Адади ҳақиқии ε (ε > 0) дода шудааст. Пайдарпайии ададҳои воқеии AK чунин муайян карда мешавад:

A1 = 2, AK = 2 + 1 / AK-1, K = 2, 3, …

Аввалин адади К-ро ёбед, ки шарти |AK - AK-1| <ε иҷро шавад.

10.29. Адади ҳақиқии ε (ε > 0) дода шудааст. Пайдарпайии ададҳои воқеии AK чунин муайян карда мешавад:

A1 = 1, A2 = 2, AK = (AK-2 + 2 · AK-1) / 3, K = 3, 4,….

Аввалин адади К-ро ёбед, ки шарти |AK - AK-1| <ε иҷро шавад.

10.30. Ададҳои мусбии A, B, C дода шудаанд. Дар росткунҷаи андозаи A × B шумораи максималии квадрати тарафаш C гузошта шудааст. Шумораи квадратҳоро, ки дар росткунҷа ҷойгир шудаанд, ёбед. Амалҳои зарб ва тақсимро истифода набаред.
