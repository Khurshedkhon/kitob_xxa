11. МАСЪАЛАҲО БО ПАЙДАРПАЙИИ ДОДАҲО

 

11.1. Даҳто ададҳои ҳақиқӣ дода шудаанд. Ҳосили ҷамъи онҳоро ёбед.

11.2. Даҳто ададҳои ҳақиқӣ дода шудаанд. Ҳосили зарби онҳоро ёбед.

11.3. Даҳто ададҳои ҳақиқӣ дода шудаанд. Қимати миёнаи арифметикии онҳоро ёбед.

11.4. Адади бутуни N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд. Ҳосили ҷамъ ва зарби маҷмӯи ададҳои додашударо чоп кунед.

11.5. Адади бутуни N ва маҷмӯи N ададҳои мусбати ҳақиқӣ дода шудаанд. Ҳосили ҷамъи қисмҳои бутуни ҳамаи ададҳоро ёбед.

11.6. Адади бутуни N ва маҷмӯи N ададҳои мусбати ҳақиқӣ дода шудаанд. Ҳосили ҷамъи қисмҳои касрии ҳамаи ададҳоро ёбед.

11.7. Адади бутуни N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд. Ҳосили ҷамъи қисмҳои яклухт карда шудаи ҳамаи ададҳоро ёбед.

11.8. Адади бутуни N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд. Миқдор ва ҳамаи ададҳои ҷуфтро аз маҷмӯи додашуда, аз рӯи тартиби ҷойгиршавиашон чоп кунед.

11.9. Адади бутуни N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд. Миқдор ва ҳамаи ададҳои тоқро аз маҷмӯи додашуда, аз рӯи тартиби ҷойгиршавиашон чоп кунед.

11.10. Адади бутуни N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд. Агар маҷмӯъ пурра аз ададҳои мусбат иборат бошад, он гоҳ сатри “ҳа”, вагарна сатри “не”-ро чоп кунед.

11.11. Ададҳои бутуни K, N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд. Агар ададҳои маҷмӯъ пурра аз адади K хурд бошад, он гоҳ сатри “ҳа”, вагарна сатри “не”-ро чоп кунед

11.12. Маҷмӯи ададҳои бутуни ғайринулӣ  дода шудааст, аломати ба итмом расидани маҷмӯъ адади 0 мебошад. Шумораи ададҳоро дар маҷмӯъ чоп кунед.

11.13. Маҷмӯи ададҳои бутуни ғайринулӣ  дода шудааст, аломати ба итмом расидани маҷмӯъ адади 0 мебошад. Ҳосили ҷамъи ададҳои мусбатро дар маҷмӯъ ёбед.

11.14. Адади бутуни K ва маҷмӯи ададҳои бутуни ғайринулӣ дода шудааст, аломати ба итмом расидани маҷмӯъ адади 0 мебошад. Шумораи ададҳои маҷмуъро ёбед, ки аз адади K калон бошад. Агар чунин ададҳо набошанд, он гоҳ 0-ро чоп кунед.

11.15. Адади бутуни K ва маҷмӯи ададҳои бутуни ғайринулӣ дода шудааст, аломати ба итмом расидани маҷмӯъ адади 0 мебошад. Аввалин адади маҷмӯъаро ёбед, ки аз адади K калон бошад. Агар чунин ададҳо набошанд, он гоҳ 0-ро чоп кунед.

11.16. Ададҳои бутуни K, N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд: A1, A2,…, AN. Дараҷаи K-уми ададҳоро чоп кунед.

A1K, A2K,…, ANK

11.17. Адади N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд: A1, A2,…, AN. Ададҳои зеринро чоп кунед.

A1, A22,…, AN-1 (N - 1), ANN

11.18. Адади N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд: A1, A2,…, AN. Ададҳои зеринро чоп кунед.

A1N, A2 (N – 1),…, AN - 12, AN.

11.19. Ададҳои K, N ва инчунин K-то маҷмӯи ададҳои аз N ададҳои бутун дода шудаанд. Ҳосили ҷамъи ҳамаи элементҳои ба ин маҷмӯаҳо дохилшударо ҳисоб кунед.

11.20. Ададҳои K, N ва инчунин K-то маҷмӯи ададҳои аз N ададҳои бутун дода шудаанд. Ҳосили ҷамъи ҳамаи элементҳои ба ин маҷмӯаҳо дохилшударо дар алоҳидагӣ ҳисоб кунед.

11.21. Ададҳои K, N ва инчунин K-то маҷмӯи ададҳои аз N ададҳои бутун дода шудаанд. Шумораи маҷмӯаҳоро, ки дар  он адади 2 вомехӯрад, ёбед. Агар чунин маҷмӯъ набошанд, он гоҳ 0-ро чоп кунед.

11.22. Ададҳои K, N ва инчунин K-то маҷмӯи ададҳои аз N ададҳои бутун дода шудаанд. Шумораи маҷмӯаҳоро, ки бо адади 2 оғоз мешавад, ёбед. Агар чунин маҷмӯъ набошанд, он гоҳ 0-ро чоп кунед.

11.23. Ададҳои K, N ва инчунин K-то маҷмӯи ададҳои аз N ададҳои бутун дода шудаанд. Шумораи маҷмӯаҳоро, ки бо адади 2 ба охир мерасад, ёбед. Агар чунин маҷмӯъ набошанд, он гоҳ 0-ро чоп кунед.

11.24. Ададҳои K, N ва инчунин K-то маҷмӯи ададҳои аз N ададҳои бутун дода шудаанд. Ҳосили ҷамъи элементҳои маҷмӯаҳоро, ки дар он адади 2 вомехӯрад, дар алоҳидагӣ чоп кунед. Агар чунин маҷмӯъ набошанд, он гоҳ 0-ро чоп кунед.

11.25. Адади бутуни K ва K-то маҷмӯи ададҳои бутуни ғайринулӣ дода шудаанд. Аломати ба охир расидани ҳар як маҷмӯъ адади 0 мебошад. Шумораи элементҳои ҳар як маҷмӯаро чоп кунед.

11.26. Адади бутуни K ва K-то маҷмӯи ададҳои бутуни ғайринулӣ дода шудаанд. Аломати ба охир расидани ҳар як маҷмӯъ адади 0 мебошад. Шумораи маҷмӯаро, ки элементҳояшон аз рӯи зиёдшавӣ ба тартиб оварда шудаанд, чоп кунед.

11.27. Адади бутуни K ва K-то маҷмӯи ададҳои бутуни ғайринулӣ дода шудаанд. Аломати ба охир расидани ҳар як маҷмӯъ адади 0 мебошад. Шумораи маҷмӯаро, ки элементҳояшон аз рӯи камшавӣ ба тартиб оварда шудаанд, чоп кунед.

11.28. Адади бутуни K ва K-то маҷмӯи ададҳои бутуни ғайринулӣ дода шудаанд. Ҳар як маҷмӯа ҳадди ақал ду элементро дар бар мегирад. Аломати ба охир расидани ҳар як маҷмӯъ адади 0 мебошад. Барои ҳар як маҷмӯъ амалҳои зеринро иҷро кунед: агар элементҳои маҷмӯъ афзуншаванда бошанд, он гоҳ натиҷаи 1; вагарна агар элементҳои маҷмӯъ камшаванда бошанд, он гоҳ натиҷаи -1; дар дигар ҳолат натиҷа 0.

11.29. Адади бутуни K ва инчунин K-то маҷмӯи ададҳои бутуни ғайринулӣ дода шудааст. Ҳар як маҷмӯа ҳадди ақал аз се элемент иборат аст. Аломати ба итмом расидани маҷмӯъ адади 0 мебошад. Шумораи маҷмӯи аррашаклро ҳисоб кунед.

11.30. Адади бутуни K ва инчунин K-то маҷмӯи ададҳои бутуни ғайринулӣ дода шудааст. Ҳар як маҷмӯа ҳадди ақал аз се элемент иборат аст. Аломати ба итмом расидани маҷмӯъ адади 0 мебошад. Барои ҳар як маҷмӯъ амалҳои зеринро иҷро кунед: агар маҷмӯъ аррашакл бошад, он гоҳ шумораи элементҳои онро бароред; вагарна, шумораи элементи аввалро, ки дандони арра нест, бароред.
