13. МАСЪАЛАҲО БАРОИ КОРКАРДИ МАССИВҲОИ ЯК ЧЕНАКДОР

 

13.1. Адади бутуни N (N > 0) дода шудааст. Массиви ададҳои бутуни андозаи N-ро ташкил диҳед, ки аз N-то ададҳои аввалини тоқи мусбат иборат бошад: 1, 3, 5, ...

13.2. Адади бутуни N (N > 0) дода шудааст. Массиви ададҳои бутуни андозаи N-ро ташкил диҳед, ки аз ададҳои ду дар дараҷаи як то N иборат бошад: 2, 4, 8, 16,….

13.3. Ададҳои бутуни A, D, N (N > 0) дода шудааст. Массиви ададҳои бутуни андозаи N-ро ташкил диҳед, ки аз N аъзои аввалини прогрессияи арифметикӣ иборат бошад:

A, A + D, A + 2·D, A + 3·D,….

13.4. Ададҳои бутуни A, D, N (N > 0) дода шудааст. Массиви ададҳои бутуни андозаи N-ро ташкил диҳед, ки аз N аъзои аввалини прогрессияи геометрӣ иборат бошад:

A, A·D, A·D2, A·D3,….

13.5. Адади бутуни N (N > 2) дода шудааст. Массиви ададҳои бутуни андозааш N-ро ташкил диҳед, ки аз N адади аввалини Фибоначчи иборат бошад:

F1 = 1, F2 = 1, FK = FK-2 + FK-1, K = 3, 4,….

13.6. Ададҳои бутуни N (N > 2), A ва B дода шудаанд. Массиви ададҳои бутуни андозааш N-ро ташкил диҳед, ки аъзои якум ба адади А, аъзои дуюм ба адади B ва ҳар як аъзои навбатӣ ба ҳосили ҷамъи аъзоҳои пешина баробар бошад.

13.7. Массиви ададҳои бутуни андозааш N дода шудааст. Аъзоҳои массивро ба тартиби баръакс чоп кунед.

13.8. Массиви ададҳои бутуни андозааш N дода шудааст. Ҳамаи ададҳои тоқи массивро аз рӯи афзуншавии рақами тартибиашон чоп кунед.

13.9. Массиви ададҳои бутуни андозааш N дода шудааст. Ҳамаи ададҳои ҷуфти массивро аз рӯи камшавии рақами тартибиашон чоп кунед.

13.10. Массиви ададҳои бутуни андозааш N дода шудааст. Ҳамаи ададҳои тоқи массивро аз рӯи афзуншавии рақами тартибиашон ва сипас ҳамаи ададҳои ҷуфти массивро аз рӯи камшавии рақами тартибиашон чоп кунед.

13.11. Массиви ададҳои бутуни A бо андозаи N ва адади бутуни K (1 ≤ K ≤ N) дода шудаанд. Ҳосили ҷамъи аъзоҳои массив, ки рақами тартибиашон ба адади K бе бақия тақсим мешаванд, ёбед.

13.12. Массиви ададҳои бутуни A бо андозаи N (N адади ҷуфт мебошад) дода шудааст. Ҳосили ҷамъи аъзоҳои массив, ки рақами тартибиашон ҷуфт аст, ёбед: A2, A4, A6,…, AN. Амали шартиро истифода набаред.

13.13. Массиви ададҳои бутуни A бо андозаи N (N адади тоқ мебошад) дода шудааст. Аъзоҳои массивро, ки рақами тартибиашон тоқ аст, аз рӯи камшавии рақами тартибӣ чоп кунед: AN, AN-2, AN-4,…, A1. Амали шартиро истифода набаред.

13.14. Массиви ададҳои бутуни A бо андозаи N дода шудааст. Аввал аъзоҳои рақами тартибиашон ҷуфт ва сипас аъзоҳои рақами тартибиашон тоқро чоп кунед:

A2, A4, A6,…, A1, A3, A5,….

Оператори шартиро истифода набаред.

13.15. Массиви ададҳои бутуни A бо андозаи N дода шудааст. Аввал аъзоҳои рақами тартибиашон тоқро аз рӯи зиёдшавӣ ва сипас аъзоҳои рақами тартибиашон ҷуфтро аз рӯи камшавӣ чоп кунед:

A1, A3, A5, ..., A6, A4, A2.

Оператори шартиро истифода набаред.

13.16. Массиви ададҳои бутуни андозааш N ва ададҳои бутуни K ва L (1 ≤ K ≤ L ≤ N) дода шудаанд. Ҳосили ҷамъи аъзоҳои массивро, ки рақами тартибиашон аз K то L мебошанд, ёбед.

13.17. Массиви ададҳои бутуни андозааш N ва ададҳои бутуни K ва L (1 ≤ K ≤ L ≤ N) дода шудаанд. Қимати миёнаи арифметикии аъзоҳои массивро, ки рақами тартибиашон аз K то L мебошанд, ёбед.

13.18. Массиви ададҳои бутуни андозааш N ва ададҳои бутуни K ва L (1 ≤ K ≤ L ≤ N) дода шудаанд. Ҳосили ҷамъи аъзоҳои массивро, ки рақами тартибиашон аз K то L мебошанд (ба истиснои аъзоҳои K-ум ва L-ум), ёбед.

13.19. Массиви ададҳои бутуни андозааш N ва ададҳои бутуни K ва L (1 ≤ K ≤ L ≤ N) дода шудаанд. Қимати миёнаи арифметикии аъзоҳои массивро, ки рақами тартибиашон аз K то L мебошанд (ба истиснои аъзоҳои K-ум ва L-ум), ёбед.

13.20. Массиви ададҳои бутуни андозааш N дода шудааст. Муайян кунед, ки оё аъзоҳои массив прогрессияи арифметикиро ташкил медиҳанд ё не. Дар ҳолати ташкил додани прогрессияи арифметикӣ қимати қадами ивазшавии прогрессияро ёбед, вагарна натиҷаи 0-ро чоп кунед.

13.21. Массиви ададҳои бутуни андозааш N дода шудааст. Муайян кунед, ки оё аъзоҳои массив прогрессияи геометриро ташкил медиҳанд ё не. Дар ҳолати ташкил додани прогрессияи геометрӣ қимати дараҷаи ивазшавии прогрессияро ёбед, вагарна натиҷаи 0-ро чоп кунед.

13.22. Массиви ададҳои бутуни андозааш N дода шудааст. Муайян кунед, ки оё аъзоҳои массив пайдарпайии ададҳои ҷуфт ва тоқро ташкил медиҳад ё не. Дар ҳолати ба талабот ҷавобгӯ набудани аъзоҳои массив натиҷаи 0-ро чоп кунед.

13.23. Массиви ададҳои бутуни андозааш N дода шудааст. Муайян кунед, ки оё аъзоҳои массив пайдарпайии ададҳои мусбат ва манфиро ташкил медиҳад ё не. Дар ҳолати ба талабот ҷавобгӯ набудани аъзоҳои массив натиҷаи 0-ро чоп кунед.

13.24. Адади бутуни N ва массиви ададҳои бутуни андозааш N дода шудааст. Ҳосили ҷамъи аъзоҳои ба 5 бе бақия тақсим шавандаро ёбед.

13.25. Адади бутуни N ва массиви ададҳои бутуни андозааш N дода шудааст. Шумораи аъзоҳои ба 7 бе бақия тақсим шавандаро ёбед.

13.26. Массиви ададҳои бутуни А ва В бо андозаи якхелаи N дода шудаанд. Қимати аъзоҳои массиви А ва В-ро бо ҳамдигар иваз кунед. Аввал аъзоҳои массиви тағйирёфтаи А ва пас аз он аъзоҳои массиви В-ро чоп кунед.

13.27. Массиви ададҳои бутуни А бо андозаи N дода шудааст. Ба массиви нави В ҳамаи аъзоҳои массиви А, ки рақами тартибиашон ҷуфт аст ва ба массиви С ҳамаи аъзоҳои массиви А, ки рақами тартибиашон тоқ аст, нависед. Амали шартиро истифода набаред.

13.28. Ду массиви А ва В бо андозаи якхела N дода шудаанд. Ба массиви нави С аъзои байни массивҳои А ва В қиматаш калонтарин, ки рақами тартибиашон якхела мебошанд, нависед.

13.29. Массиви ададҳои бутуни А бо андозаи N дода шудааст. Аъзоҳои массиви А-ро ба массиви нави B аз рӯи афзуншавии қиматашон нависед.

13.30. Массиви ададҳои бутуни А бо андозаи N дода шудааст. Аъзоҳои массиви А-ро ба массиви нави С аз рӯи камшавии қиматашон нависед.
