16. МАСЪАЛАҲО БО ҚИМАТҲОИ КАЛОНТАРИН ВА ХУРДТАРИН

 

16.1. Адади бутуни N ва маҷмӯи N ададҳои ҳақиқӣ дода шудаанд. Қимати калонтарин ва хурдтарини маҷмӯи додашударо ёбед.

16.2. Адади бутуни N ва маҷмӯи N росткунҷаи паҳлӯҳояшон бо ҷуфти ададҳои А(N), В(N)  муайян карда шудаанд, дода шудаанд. Аз байни росткунҷаҳо росткунҷаи масоҳаташ калонтаринро ёбед.

16.3. Адади бутуни N ва маҷмӯи N росткунҷаи паҳлӯҳояшон бо ҷуфти ададҳои А(N), В(N) муайян карда шудаанд, дода шудаанд. Аз байни росткунҷаҳо росткунҷаи периметраш хурдтаринро ёбед.

16.4. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Аз маҷмӯъшумораи адади хурдтаринро ёбед.

16.5. Адади бутуни N ва ҷуфти ададҳои M(N) – массаи ҷисм ва V(N) – ҳаҷми ҷисм дода шудаанд. Зичии P бо формулаи P = M / V ҳисоб карда мешавад. Аз байни N ҷисм, рақами тартиби ҷисми зиччиаш аз ҳама калонтаринро ёбед.

16.6. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Маълум аст, ки ададҳо дар маҷмӯъ такрор намеёбанд. Аз маҷмӯи додашуда, рақамҳои тартибии ададҳои калонтарин ва хурдтаринро муайян кунед.

16.7.  Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Маълум аст, ки ададҳо дар маҷмӯъ такрор намеёбанд. Аз маҷмӯи додашуда, ададҳоро аз адади якум то адади калонтарин чоп кунед.

16.8.  Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Маълум аст, ки ададҳо дар маҷмӯъ такрор намеёбанд. Аз маҷмӯи додашуда, ададҳоро аз адади хурдтарин то адади охирон чоп кунед.

16.9.  Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Маълум аст, ки ададҳо дар маҷмӯъ такрор намеёбанд. Аз маҷмӯи додашуда, ададҳои байни адади хурдтарин ва калонтарин ҷойгир шударо чоп кунед.

16.10. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Адади экстремалӣ – ин адади калонтарин ё хурдтарин ба ҳисоб меравад. Дар маҷмӯи додашуда рақами тартибии адади экстремалии аввалинро ёбед.

16.11.  Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Адади экстремалӣ – ин адади калонтарин ё хурдтарин ба ҳисоб меравад. Дар маҷмӯи додашуда рақами тартибии адади экстремалии охиронро ёбед.

16.12. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Аз маҷмӯи додашуда, хурдтарин адади мусбатро ёбед. Агар дар маҷмӯъ адади мусбат набошад, он гоҳ 0-ро чоп кунед.

16.13. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Аз маҷмӯи додашуда, калонтарин адади тоқро ёбед. Агар дар маҷмӯъ адади тоқ набошад, он гоҳ 0-ро чоп кунед.

16.14. Адади бутуни B (B > 0) ва маҷмӯи даҳ адади бутун дода шудааст. Ҳосили ҷамъ ва миқдори ададҳои маҷмӯъ, ки аз адади В калон аст ҳисоб кунед. Агар дар маҷмӯъ ададҳои аз B калон вуҷуд надошта бошанд, он гоҳ натиҷаи 0-ро чоп кунед.

16.15. Ададҳои B, C (0 <B <C) ва маҷмӯи даҳто ададҳои бутун дода шудаанд. Адади калонтаринро аз маҷмӯи дода шуда муайян кунед, ки дар фосилаи B ва C ҷойгир шуда бошад.

16.16. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Шумораи ададҳоеро, ки пеш аз аввалин адади хурдтарин ҷойгир шудаанд, ёбед.

16.17. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Шумораи ададҳоеро, ки пас аз охирон адади калонтарин ҷойгир шудаанд, ёбед.

16.18. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Шумораи ададҳоеро, ки байни аввалин ва охирон адади калонтарин ҷойгир шудаанд, ёбед. Агар дар маҷмӯъ танҳо як адади калонтарин бошад, он гоҳ натиҷаи 0-ро чоп кунед.

16.19. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Шумораи ададҳои хурдтаринро дар маҷмӯъ ёбед.

16.20. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Адади экстремалӣ – ин адади калонтарин ё хурдтарин ба ҳисоб меравад. Дар маҷмӯи додашуда шумораи умумии адади экстремалиро ёбед.

16.21. Адади бутуни N (N > 2) ва маҷмӯи N-то ададҳои бутун дода шудаанд. Қимати миёнаи арифметикии маҷмӯъро бе назардошти ададҳои хурдтарин ва калонтарин ёбед.

16.22. Адади бутуни N (N > 2) ва маҷмӯи N-то ададҳои бутун дода шудаанд. Аз маҷмӯи додашуда ду адади хурдтаринро муайян кунед. Ин ададҳоро аз рӯи зиёдшавӣ чоп кунед.

16.23. Адади бутуни N (N > 3) ва маҷмӯи N-то ададҳои бутун дода шудаанд. Аз маҷмӯи додашуда се адади калонтаринро муайян кунед. Ин ададҳоро аз рӯи камшавӣ чоп кунед.

16.24. Адади бутуни N (N > 3) ва маҷмӯи N-то ададҳои бутун дода шудаанд. Калонтарини ҳосили ҷамъи ду ададҳои ҳамсояро дар маҷмӯи додашуда ёбед.

16.25. Адади бутуни N (N > 3) ва маҷмӯи N-то ададҳои бутун дода шудаанд. Хурдтарин ҳосили ҷамъи се ададҳои ҳамсояро дар маҷмӯи додашуда ёбед.

16.26. Адади бутуни N (N > 3) ва маҷмӯи N-то ададҳои бутун дода шудаанд. Се ададҳои ҷуфти калонтаринро дар маҷмӯи додашуда ёбед. Агар дар маҷмӯъ ададҳои ҷуфт набошанд, он гоҳ натиҷаи 0-ро чоп кунед.

16.27. Адади бутуни N ва маҷмӯи N-то ададҳои бутуни 0 ва 1 дода шудаанд. Шумораи ададҳои якхела (0 ё 1), ки пайдарпайии дарозтаринро ташкил медиҳанд ёбед. Агар якчанд чунин пайдарпаӣ мавҷуд бошад, он гоҳ пайдарпайии якумро ба инобат гиред.

16.28. Адади бутуни N ва маҷмӯи N-то ададҳои бутуни 0 ва 1 дода шудаанд. Шумораи ададҳои 1, ки пайдарпайии дарозтаринро ташкил медиҳанд ёбед. Агар якчанд чунин пайдарпаӣ мавҷуд бошад, он гоҳ пайдарпайии якумро ба инобат гиред.

16.29. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Пайдарпайии дарозтарини ададҳои маҷмӯъ, ки аз рӯи зиёдшавӣ ҷойгир шудаанд, ёбед.

16.30. Адади бутуни N ва маҷмӯи N-то ададҳои бутун дода шудаанд. Пайдарпайии дарозтарини ададҳои маҷмӯъ, ки аз рӯи камшавӣ ҷойгир шудаанд, ёбед.
