17. МАСЪАЛАҲО БАРОИ КОРКАРДИ
ДОДАҲОИ ФАЙЛ

 

17.1. Сатри S дода шудааст. Муайян кунед, ки бо сатри S файлро дуруст номгузорӣ кардан мумкин аст ё не.

17.2. Адади бутуни N (N > 1) дода шудааст. Файлеро созед, ки дар ҳар сатри он N-то ададҳои мусбати ҷуфт сабт шуда бошанд (2, 4, ...).

17.3. Ададҳои бутуни А ва D дода шудаанд. Файлеро созед, ки дар ҳар сатри он даҳто аъзои прогрессияи арифметикӣ сабт шуда бошанд Аъзои якуми прогрессия ба А ва фарқи байни аъзоҳо ба D баробар аст:

A, A + D, A + 2 D, A + 3 D,…

17.4. Номи чор файл оварда шудааст. Шумораи файлҳоро бо номҳои муайяншуда дар каталоги ҷорӣ сабт шудаанд, ёбед.

17.5. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Шумораи ададҳои дар файли додашуда мавҷудбударо ёбед.

17.6. Адади бутуни K ва файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Адади дар сатри K-уми файл ҷойгир шударо чоп кунед (рақамгузорӣ аз 1 оғоз карда мешаванд). Агар чунин адад вуҷуд надошта бошад, онгоҳ натиҷаи 0-ро чоп кунед.

17.7. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Адади дар сатри якум ва охирони файл ҷойгир шударо чоп кунед.

17.8. Ду файл дода шудааст. Дар ҳар сатри файли якум ададҳои бутун сабт карда шудаанд. Файли дуюм холӣ мебошад. Маълумоти файли якумро ба файли дуюм нависед.

17.9. Ду файл дода шудааст. Дар ҳар сатри файли якум ададҳои бутун сабт карда шудаанд. Файли дуюм холӣ мебошад. Аз файли якум ба ғайр аз адади якум ва охиронаш ададҳоро ба файли дуюм нависед.

17.10. Файл дода шудааст, ки дар ҳар сатри он зиёда аз 50-то ададҳои бутун сабт шудаанд. Аз файли додашуда ададҳои аз 50-то зиёдро хориҷ кунед.

17.11. Файл дода шудааст, ки миқдори ҷуфти ададҳои бутунро дар бар мегирад. Нимаи дуюми ададҳоро аз файл хориҷ кунед.

17.12. Файл дода шудааст, ки дар ҳар сатри он зиёда аз 50-то ададҳои бутун сабт шудаанд. Аз файли додашуда 50-то адади аввалро хориҷ кунед.

17.13. Файл дода шудааст, ки миқдори ҷуфти ададҳои бутунро дар бар мегирад. Нимаи аввали ададҳоро аз файл хориҷ кунед.

17.14. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Ҳама ададҳои ҷуфтро аз файл хориҷ кунед.

17.15. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Ҳама ададҳои манфиро аз файл хориҷ кунед.

17.16. Файл дода шудааст, ки дар ҳар сатри он аз 50-то камтар ададҳои бутун сабт шудаанд. Ба файли додашуда миқдори сатрҳоро то 50-то бо илова кардани адади 0 ба ҳар сатри нав зиёд кунед.

17.17. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Ҳаҷми файлро бо роҳи аз охири файл аз нав навиштани ададҳои файл ду маротиба зиёд кунед.

17.18. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Ҳаҷми файлро бо роҳи аз охири файл аз нав навиштани ададҳои файл ба таври баръакс ду маротиба зиёд кунед.

17.19. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Ҳаҷми файлро бо роҳи такрор намудани ададҳои дар сатри рақами тартибиаш тоқ ҷойгиршуда, тағйир диҳед.

17.20. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Ҳаҷми файлро бо роҳи ду маротиба такрор намудани ададҳои дар сатри рақами тартибиашон дар фосилаи аз 5 то 10 ҷойгир буда, иваз кунед.

17.21. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Ҳар як адади ҷуфти файлро бо сифр иваз кунед.

17.22. Файл дода шудааст, ки дар ҳар сатри он ададҳои бутун сабт шудаанд. Ҳар як адади мусбати файлро бо ду сифри пайдарпай иваз кунед.

17.23. Се файл дода шудаанд, ки ба миқдори якхела дар ҳар сатр ададҳои бутунро дар бар мегиранд. Ба файли нави чорум аз файлҳо ададҳоро аз рӯи тартиби зерин сабт кунед: якум сатри файли якум, якум сатри файли дуюм, якум сатри файли сеюм, дуюм сатри файли якум, дуюм сатри файли дуюм, дуюм сатри файли сеюм, ва ғайра. 

17.24. Се файл дода шудаанд, ки ба миқдори гуногун дар ҳар сатр ададҳои бутунро дар бар мегиранд. Ба файли нави чорум аз файлҳо ададҳоро аз рӯи тартиби зерин сабт кунед: аввал маълумоти файли миқдори сатрҳояш аз ҳама камтарин, сипас маълумоти файли миқдори сатрҳояш аз ҳама зиёдтарин ва дар охир маълумоти файли боқимонда.

17.25. Ду файл дода шудаанд, ки ба миқдори гуногун дар ҳар сатр ададҳои бутунро аз рӯи афзуншавӣ дар бар мегиранд. Ба файли нави сеюм аз файлҳои якум ва дуюм ададҳоро чунин сабт кунед, ки онҳо низ аз рӯи афзуншавӣ ҷойгир шаванд.

17.26. Ду файл дода шудаанд, ки ба миқдори гуногун дар ҳар сатр ададҳои бутунро аз рӯи афзуншавӣ дар бар мегиранд. Ба файли нави сеюм аз файлҳои якум ва дуюм ададҳоро чунин сабт кунед, ки онҳо низ аз рӯи камшавӣ ҷойгир шаванд.

17.27. Файл дода шудааст, ки дар ҳар сатри он калимаҳо сабт шудаанд. Ба файли нав калимаҳои дарозиашон кӯтоҳтаринро аз рӯи тартиби вохӯриашон нависед.

17.28. Файл дода шудааст, ки дар ҳар сатри он калимаҳо сабт шудаанд. Ба файли нав калимаҳои дарозиашон дарозтаринро аз рӯи тартиби вохӯриашон баръакс нависед.

17.29. Файл дода шудааст, ки дар ҳар сатри он калимаҳо сабт шудаанд. Ба файли нав калимаҳои аз рӯи алифбо ба тартиби зиёдшавӣ нависед.

17.30. Файл дода шудааст, ки дар ҳар сатри он калимаҳо сабт шудаанд. Ба файли нав калимаҳоро аз рӯи тартиби зиёдшавии дарозиашон нависед.
