25. МАСЪАЛАҲОИ ТАЪМИНИ БЕХАТАРИИ ДОДАҲО ВА КРИПТОГРАФИЯ

 

25.1. Матни додашударо ба формати HEX табдил диҳед.

25.2. Матни додашударо ба формати BASE64 табдил диҳед.

25.3. Барнома барои барқарор намудани матни шифршудаи формати BASE64 тартиб диҳед.

25.4. Адади калони бутун дода шудааст. Бо истифода аз пакети OpenSSL квадрати ин ададро ҳисоб намоед.

25.5. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI ба матни додашуда аз рӯи алгоритми MD5 хеш ҳисоб намоед.

25.6. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI ба матни додашуда аз рӯи алгоритми HMACSHA512 хеш ҳисоб намоед.

25.7. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI ба матни додашуда аз рӯи алгоритми SHA1 хеш ҳисоб намоед.

25.8. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI ба матни додашуда аз рӯи алгоритми SHA256 хеш ҳисоб намоед.

25.9. Адади тасодуфӣ аз рӯи формулаи зерин муайян карда мешавад:

Xk+1 = (a*Xk + c) mod m

дар ин ҷо a, c, m коэффисентҳои бутун буда, Xk қимати аввала мебошад. Аз рӯи формула адади тасодуфӣ ҳосил намоед.

25.10. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI псевдо-рамз бо дарозии додашуда ҳосил намоед.

25.11. Барои шифронӣ намудани матни забони англисӣ ҳарфҳо мувофиқи тартиби алфавитҳои русӣ ва англисӣ  истифода мешаванд (a,b,c,d… - а,б,в,г..). Барнома барои шифр намудани матни англисӣ бо иваз намудани ҳарфҳои мувофиқ тартиб диҳед.

25.12. Матн ва рамз дода шудааст. Бо истифода аз пакети OpenSSL матни додашударо аз рӯи алгоритми Blowfish шифр намоед.

25.13. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI матни додашударо аз рӯи алгоритми DES шифр намоед. Калиди шифр дода мешавад. Фарқи реҷаҳои алгоритми DES-ро шарҳ диҳед.

25.14. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI матни додашударо аз рӯи алгоритми 3DES шифр намоед. Калиди шифр дода мешавад.

25.15. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI матни додашударо аз рӯи алгоритми AES (Rijndael) шифр намоед. Калиди шифр дода мешавад.

25.16. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI ҷуфти калиди шифр барои алгоритми RSA ҳосил намоед.

25.17. RSA калиди пӯшида дар формати PEM дода шудааст. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI барнома барои табдилдиҳии калид барои истифода дар CryptoAPI таҳия намоед.

25.18. RSA калиди кушода дар формати PEM дода шудааст. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI барнома барои табдилдиҳии калид барои истифода дар CryptoAPI таҳия намоед.

25.19. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI рамзи додашуда аз рӯи алгоритми RSA шифр карда шавад. Калиди пӯшида дода шудааст.

25.20. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI рамз аз рӯи алгоритми RSA барқарор карда шавад. Рамзи шифршуда ва калиди кушода дода шудаанд.

25.21. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI ба матни додашуда имзои электронии рақамӣ аз рӯи алгоритми RSA ҳосил намоед. Калиди пӯшида дода шудааст.

25.22. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI имзои электронии рақамии матни додашуда санҷида шавад. Имзои рақамӣ аз рӯи алгоритми RSA ҳосил шуда, калиди кушода дода шудааст.

25.23. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI сертификати  X.509 ҳосил намоед. Маълумотҳои сертификат ихтиёрист. 

25.24. Сертификати X.509 ба намуди файл дода шудааст. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI барнома барои инъикоси маълумотҳои ин сертификат тартиб диҳед.

25.25. Сертификати X.509 ба намуди файл дода шудааст. Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI калиди кушодаро аз сертификат ҳосил намоед.

25.26. Матн дода шудааст. Барнома барои фишурдани матн бо истифода аз пакети zlib тартиб диҳед.

25.27. Матн дода шудааст. Барнома барои фишурдани матн аз рӯи алгоритми Хаффман тартиб диҳед. Барои коркарди барнома аз пакетҳои озодасос истифода бурда метавонед.

25.28. Матн дода шудааст. Барнома барои фишурдани матн аз рӯи алгоритм Шеннон тартиб диҳед. Барои коркарди барнома аз пакетҳои озодасос истифода бурда метавонед.

25.29. Сертификати намуди X.509 формати PEM ва калиди пӯшидаи он дода шудааст.  Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI сертификати намуди PKCS # 12 ҳосил намоед.

25.30. Сертификати намуди X.509 формати PEM дода шудааст.  Бо истифода аз пакети OpenSSL ё Microsoft CryptoAPI сертификати формати DER ҳосил намоед.
