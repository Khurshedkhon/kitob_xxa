Банди 2. Бо истифодаи имкониятҳои забони SQL масъалаҳои пешниҳод шударо, пайдарпай иҷро кунед. Ҳалли масъалаҳо аз муҳити системаи идораи манбаи додаҳо вобастагӣ надорад.

 

28.1. CREATE DATABASE, DROP DATABASE

Ø Манбаи додаҳои DBase_Korgar-ро созед.

Ø Манбаи додаҳои DBase_Korgar-ро хориҷ кунед.

Ø Манбаи додаҳои DBase_Korkhona-ро созед.

 

28.2. CREATE TABLE, RENAME TABLE, DROP TABLE

Ø Ҷадвали tblVazifa-ро созед.

Ø Ҷадвали tblKorgaron-ро созед.

Ø Ҷадвали tblLoiha-ро созед.

Ø Ҷадвали tblParvandaKorgar-ро созед ва бо ҷадвали tblKorgaron бо муносибати намуди як бар як пайваст кунед.

Ø Ҷадвали tblIjroVazifa-ро созед, онро бо ҷадвали tblVazifa бо муносибати намуди бисёр бар як ва бо ҷадвали tblKorgaron бо муносибати намуди бисёр бар як пайваст кунед.

Ø Ҷадвали tblKorkardLoiha-ро созед, онро бо ҷадвали tblLoiha бо муносибати намуди бисёр бар як ва бо ҷадвали tblKorgaron бо муносибати намуди бисёр бар як пайваст кунед.

 

28.3. ALTER TABLE, ADD COLUMN, RENAME COLUMN, ALTER COLUMN, DROP COLUMN

Ø Ба ҷадвали tblKorgaron сутуни стажи корӣ (stajKorgar) бо намуди додаҳои сатри кӯтоҳ илова кунед.

Ø Дар ҷадвали tblKorgaron номи сутуни стажи кориро (stajKorgar) ба собиқаи корӣ (sobiqaKorgar) иваз кунед.

Ø Дар ҷадвали tblKorgaron намуди додаҳои собиқаи кориро (sobiqaKorgar) ба намуди додаҳои адади бутун тағйир диҳед.

Ø Аз ҷадвали tblKorgaron сутуни собиқаи кориро (sobiqaKorgar) хориҷ кунед.

 

28.4. INSERT

Ø Ба ҷадвалҳои tblVazifa додаҳоро ворид кунед.

Ø Ба ҷадвалҳои tblKorgaron додаҳоро ворид кунед.

Ø Ба ҷадвалҳои tblLoiha додаҳоро ворид кунед.

Ø Ба ҷадвали tblParvandaKorgar додаҳоро ворид кунед.

Ø Ба ҷадвалҳои tblIjroVazifa додаҳоро ворид кунед.

Ø Ба ҷадвалҳои tblKorkardLoiha додаҳоро ворид кунед.

 

28.5. UPDATE, WHERE

Ø Дар ҷадвали tblLoiha арзиши ҳар як лоиҳаро 20% зиёд кунед.

Ø Дар ҷадвали tblLoiha арзиши лоиҳаи рақами 3-юмро ба 10000 кам кунед.

Ø Дар ҷадвали tblKorgaron рақами телефони Махкамова  Ситораро бо рақами 927878787 иваз кунед.

Ø Дар ҷадвали tblKorkardLoiha ҳолати иҷроиши лоиҳаҳое, ки “Иҷро шуда истодааст” ба ҳолаи “Иҷро карда шуд” иваз кунед.

 

28.6. DELETE, WHERE

Ø Аз ҷадвали tblLoiha додаҳоро хориҷ кунед.

Ø Аз ҷадвали tblKorkardLoiha додаҳоро хориҷ кунед.

Ø Аз ҷадвали tblKorgaron додаҳои коргари рақами тартибиаш 5-умро хориҷ кунед.

Ø Аз ҷадвали tblVazifa вазифаи муҳосибро хориҷ кунед.

 

28.7. TRUNCATE

Ø Додаҳои ҷадвали tblVazifa-ро тоза кунед.

Ø Додаҳои ҷадвали tblKorgaron-ро тоза кунед.

Ø Додаҳои манбаи додаҳои DBaseKorkhona-ро тоза кунед.

Эзоҳ. Масъалаҳои банди фармони INSERT-ро аз нав ба кор дароред.

 

28.8. SELECT, WHERE – интихоб аз як ҷадвал

Ø Аз ҷадвали tblVazifa ҳамаи додаҳоро интихоб кунед.

Ø Аз ҷадвали tblKorgaron ҳамаи додаҳоро интихоб кунед.

Ø Аз ҷадвали tblLoiha ҳамаи додаҳоро интихоб кунед.

Ø Аз ҷадвали tblKorgaron додаҳои насаб, ном ва санаи таваллуди коргаронро интихоб кунед.

Ø Аз ҷадвали tblLoiha додаҳои ном ва арзиши иҷрои лоиҳаро интихоб кунед.

Ø Аз ҷадвали tblKorgaron додаҳои коргарони ҷинсашон занро интихоб кунед.

Ø Аз ҷадвали tblLoiha додаҳои лоиҳаҳое, ки арзиши иҷроишашон аз 500000 зиёд аст, интихоб кунед.

Ø Аз ҷадвали tblKorgaron додаҳои насаб, ном ва суроғаи мардҳоро интихоб кунед.

Ø Аз ҷадвали tblLoiha ҳамаи додаҳои лоиҳаҳои рақами тартибиашон 1-ум ва 4-умро интихоб кунед.

 

28.9.   SELECT, JOIN, WHERE – интихоб аз якчанд ҷадвал

Ø Аз ҷадвалҳои tblKorgaron ва tblParvandaKorgar ҳамаи додаҳоро интихоб кунед.

Ø Аз ҷадвали tblKorgaron додаҳои насаб ва номи коргар ва аз ҷадвали tblParvandaKorgar додаҳои ҳолати оилавӣ ва касби коргарро интихоб кунед.
