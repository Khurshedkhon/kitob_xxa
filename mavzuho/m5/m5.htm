5. МАСЪАЛАҲОИ ОДДӢ

 

5.1 Тарафи майдони квадратшакл а дода шудааст. Қимати периметр яъне, ҳудуди онро P = 4a ёбед.

5.2 Тарафи майдони квадратшакл а дода шудааст. Масоҳати онро S = a2 ёбед.

5.3 Тарафҳои росткунҷа a ва b дода шудаанд. Масоҳати S = a · b ва периметр P = 2 (a + b) -ро ёбед.

5.4 Диаметри доира d дода шудааст. Дарозии онро L = π · d ёбед. Қимати 3.14 -ро барои π истифода баред.

5.5 Дарозии канори куб а дода шудааст. Ҳаҷми куби V = a3 ва масоҳати онро S = 6 · a2 ёбед.

5.6 Дарозии теғаҳои параллелепипеди росткунҷа дода шудааст a, b, c. Ҳаҷми онро V = a · b · c ва майдони сатҳи онро S = ​​2 (a · b + b · c + a · c) ёбед.

5.7 Радиуси додашудаи R-ро истифода бурда, дарозии доираи L ва масоҳат S-ро ёбед: L = 2 π R, S = π R2. Қимати 3.14 -ро барои π истифода баред.

5.8 Ду адади а ва b дода шудаанд. Қимати миёнаи арифметикии онҳоро ёбед: (a + b) / 2.

5.9 Ду ададҳои ғайриманфии a ва b дода шудаанд. Қимати миёнаи геометрии онҳоро, яъне решаи квадратии ҳосили зарби онҳоро ёбед: (а · b) 1/2.

5.10 Катетҳои секунҷаи росткунҷа a ва b дода шудаанд. Гипотенузаи он с ва периметри P -ро ёбед: c = (a2 + b2)1/2,
P = a + b + c.

5.11 Ду давра бо маркази умумӣ ва радиусҳои R1 ва R2 (R1> R2) дода шудаанд. Масоҳати давраҳои S1 ва S2 ва инчунин масоҳати S3 ҳалқаро ёбед, ки радиуси давраи беруниаш ба R1 ва радиуси давраи дохилиаш ба R2 мувофиқан баробар аст:
S1 = π (R1)2, S2 = π (R2)2, S3 = S1 - S2. Қимати 3.14 -ро барои π истифода баред.

5.12 Дарозии давра дода шудааст: L = 2 · π · R. Дарозии радиуси он R ва масоҳати S-и доираи бо ин доира маҳдудро ҳисоб кунед S = π · R2. Қимати 3.14 -ро барои π истифода баред.

5.13 Масоҳати давра S дода шудааст. Диаметри он D ва дарозии L давраро, бо назардошти L = π · D, S = π · D2 / 4 ёбед. Қимати 3.14 -ро барои π истифода баред.

5.14 Масофаи байни ду нуқтаро бо қимати координатаҳои додашудаи x1 ва x2 дар тири ададӣ ёбед: | x2 - x1 |.

5.15 Се нуқтаи A, B, C дар тири ададӣ дода шудаанд. Дарозии порчаҳои AC ва BC ва ҳосили ҷамъи онҳоро ёбед.

5.16 Дар тири ададӣ се нуқтаи A, B, C дода шудаанд. Нуқтаи C дар байни нуқтаҳои A ва B ҷойгир аст. Қимати дарозии порчаҳои AC ва BC -ро ёбед.

5.17 Координатаҳои ду қуллаи муқобили росткунҷа дода шудаанд: (x1, y1), (x2, y2). Қимати периметр ва масоҳати росткунҷаи додашударо ёбед.

5.18 Масофаи байни ду нуқтаро бо координатаҳои (x1, y1) ва (x2, y2) дар ҳамворӣ ёбед. Масофа бо формулаи зерин ҳисоб карда мешавад: ((x2 - x1) 2 + (y2 - y1) 2 ) 1/2.

5.19 Координатаҳои се қуллаи секунҷа дода шудаанд: (x1, y1), (x2, y2), (x3, y3). Бо истифода аз формулаи масофаи байни ду нуқта периметр ва масоҳати секунҷаро ёбед (нигаред ба вазифаи гузашта). Барои ёфтани масоҳати секунҷа бо паҳлӯҳои a, b, c формулаи Ҳеронро истифода баред: S = (p·(p - a)·(p - b)·(p - c))1/2, ки дар он p = (a + b + c) / 2 нимпериметр мебошад.

5.20 Қиматҳои тағйирёбандаҳои A ва B -ро бо ҳамдигар иваз кунед ва қиматҳои навро барои A ва B чоп кунед.

5.21 Қиматҳои тағйирёбандаҳои A, B, C дода шудаанд.  Қиматҳои онҳоро бо роҳи интиқол додани қиматҳои A ба B, B ба C, C ба A тағйир диҳед. Қимати нави тағйирёбандаҳои A, B, C -ро чоп кунед.

5.22 Қиматҳои тағйирёбандаҳои A, B, C дода шудаанд.  Қиматҳои онҳоро бо роҳи интиқол додани қиматҳои A ба C, C ба B, B ба A тағйир диҳед. Қимати нави тағйирёбандаҳои A, B, C -ро чоп кунед.

5.23 Қимати ададии А дода шудааст. Бо ёрии танҳо як тағйирёбанда ва се амали зарб пайдарпай қиматҳои A2, A4, A8 -ро ёбед. Ҳамаи қиматҳои ҳосил шударо чоп кунед.

5.24 Қимати ададии А дода шудааст. Бо ёрии танҳо ду тағйирёбанда ва панҷ амали зарб пайдарпай қиматҳои A2, A3, A5, A10, A15 -ро ёбед. Ҳамаи қиматҳои ҳосил шударо чоп кунед.

5.25 Кунҷи α бо градус дода мешавад (0 < α <360). Бо назардошти он, ки 180 ° = π радианро ифода мекунад, қимати ҳамон як кунҷро дар радианҳо муайян кунед. Қимати 3.14 -ро барои π истифода баред.

5.26 Қимати кунҷи α бо радианҳо (0 < α <2 · π) дода шудааст. Бо назардошти он, ки 180 ° = π радианро ифода мекунад,  қимати ҳамон як кунҷро бо градус муайян кунед. Қимати 3.14 -ро барои π истифода баред.

5.27 Ҳарорати ҳаво T бо дараҷаи Фаренгейт дода шудааст. Қимати ҳамон як ҳароратро бо дараҷаи Селсий муайян кунед. Ҳарорати ТС ва ҳарорати Фаренгейт TF бо алоқамандии зерин ифода карда мешавад: TC = (TF - 32) 5/9.

5.28 Ҳарорат T бо дараҷаи Селсий дода шудааст. Қимати ҳамон як ҳароратро бо дараҷаи Фаренгейт муайян кунед. Ҳарорати ТС ва ҳарорати Фаренгейт TF бо алоқамандии зерин ифода карда мешавад: TC = (TF - 32) 5/9.

5.29 Муодилаи хаттии Ax + B = 0-ро аз рӯи қиматҳои коэффитсиентҳои A ва B ҳал кунед (коэффитсиенти A ба 0 баробар нест).

5.30 Ададҳои A1, B1, C1, A2, B2, C2 дода шудаанд. Ҳалли системаи муодилаҳои хаттии шакли зеринро ёбед

A1 x + B1 y = C1

A2 x + B2 y = C2
