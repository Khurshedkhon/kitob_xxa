9. МАСЪАЛАҲО БО ИСТИФОДАИ ТАКРОРШАВӢ БО ҲИСОБКУНАК

 

9.1. Ададҳои бутуни K ва N (N> 0) дода шудаанд. Адади K-ро N маротиба чоп кунед.

9.2. Ададҳои бутуни A ва B дода шудаанд (A<B). Ҳамаи ададҳои дар байни A ва B ҷойгиршударо аз рӯи зиёдшавӣ чоп кунед.

9.3. Ададҳои бутуни A ва B дода шудаанд (A<B). Ҳамаи ададҳои дар байни A ва B ҷойгиршударо аз рӯи камшавӣ чоп кунед.

9.4. Адади эҳтимолӣ дода шудааст, ки  нархи 1 кг шириниро ифода мекунад. Нархи 1, 2,…, 10 кг шириниро чоп кунед.

9.5. Адади эҳтимолӣ дода шудааст, ки  нархи 1 кг шириниро ифода мекунад. Нархи 0.1, 0.2, … , 1 кг шириниро чоп кунед.

9.6. Адади эҳтимолӣ дода шудааст, ки  нархи 1 кг шириниро ифода мекунад. Нархи 1.2, 1.4,…, 5 кг шириниро чоп кунед.

9.7. Ду адади бутуни A ва B дода шудааст (A<B). Ҳосили ҷамъи ҳамаи ададҳои аз А то В-ро ҳисоб кунед.

9.8. Ду адади бутуни A ва B дода шудааст (A<B). Миқдори ҳамаи ададҳои аз А то В-ро ёбед.

9.9. Ду адади A ва B дода шудааст (A<B). Ҷамъи квадратҳои ҳамаи ададҳои аз А то В-ро ҳисоб кунед. Ададҳои A ва B-ро низ ба инобат гиред.

9.10. Як адади N (> 0) дода шудааст. Ҳисоб кунед:

1 + 1/2 + 1/3 +… + 1 / N

9.11. Як адади N (> 0) дода шудааст. Ҳисоб кунед:

N2 + (N + 1)2 + (N + 2)2 +… + (2 N)2

9.12. Як адади N (> 0) дода шудааст. Ҳисоб кунед:

1.1 · 1.2 · 1.3 · ... · N

9.13. Як адади N (> 0) дода шудааст. Бе истифодаи амали шартӣ ҳисоб кунед:

1.1 - 1.2 + 1.3 - ... N

 

9.14. Як адади N (> 0) дода шудааст. Бо истифода аз формулаи зерин квадрати адади N-ро ёбед:

N2 = 1 + 3 + 5 +… + (2 · N - 1).

9.15. Адади эҳтимолии A ва бутуни N (> 0) дода шудаанд. Қимати A-ро ба дараҷаи N бардоред.

AN = A · A ·… · A

9.16. Адади эҳтимолии A ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, ҳамаи дараҷаҳои А-ро аз 1 то N чоп кунед.

A1, A2, ... , AN

9.17. Адади эҳтимолии A ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, ҳосили ҷамъро ёбед.

1 + A + A2 + A3 + … + AN.

9.18. Адади эҳтимолии A ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, қимати ифодаро ёбед.

1 - A + A2 - A3 + … + (−1)·N · AN.

9.19. Адади бутуни N (N>0) дода шудааст. Ҳисоб кунед.

N! = 1 · 2 ·… · N. (N-факториал).

9.20. Адади бутуни N (N>0) дода шудааст. Бо истифода аз як давр ҳисоб кунед.

N! = 1 · 2 ·… · N

1! + 2! + 3! +… + N!

9.21. Адади бутуни N (N>0) дода шудааст. Бо истифода аз як давр қимати ифодаро ҳисоб кунед.

1 + 1 / (1!) + 1 / (2!) + 1 / (3!) +… + 1 / (N!)

9.22. Адади воқеии X ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, қимати ифодаро ёбед.

1 + X + X2 / (2!) +… + XN / (N!)

9.23. Адади воқеии X ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, қимати ифодаро ёбед.

X - X3 / (3!) + X5 / (5!) -… + (-1)·N·X2N + 1 / ((2·N + 1)!)

9.24. Адади воқеии X ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, қимати ифодаро ёбед.

1 - X2 / (2!) + X4 / (4!) -… + (-1)·N·X2N / ((2·N)!)

9.25. Адади воқеии X (|X|<1)ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, қимати ифодаро ёбед.

X - X2 / 2 + X3 / 3 -… + (-1) (N-1) · XN / N.

9.26. Адади воқеии X (|X|<1)ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, қимати ифодаро ёбед.

X - X3 / 3 + X5 / 5 -… + (-1) N · X2·N+1 / (2 · N + 1).

9.27. Адади воқеии X (|X|<1)ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, қимати ифодаро ёбед.

X + 1 · X3 / (2 · 3) + 1 · 3 · X5 / (2 · 4 · 5) + ... +

+ 1 · 3 ·… · (2 ​​· N - 1) · X2 · N + 1 / (2 · 4 ·… · (2 ​​· N) · (2 ​​· N + 1)).

9.28. Адади воқеии X (|X|<1)ва бутуни N (> 0) дода шудаанд. Бо истифода аз як давр, қимати ифодаро ёбед.

1 + X / 2 - 1 X2 / (2·4) + 1·3·X3 / (2·4·6) - … +

+ (−1) N - 1 · 1 · 3 ·… · (2 ​​· N - 3) · XN / (2 · 4 ·… · (2 ​​· N)).

9.29. Адади бутуни N (N > 0) дода шудааст. Пайдарпайии ададҳои AK чунин муайян карда мешавад.

A0 = 2, AK = 2 + 1 / AK - 1, K = 1, 2,….

Пайдарпаи A1, A2,…, AN -ро чоп кунед.

9.30. Адади бутуни N (N > 0) дода шудааст. Пайдарпайии ададҳои Фибоначчи FK (адади бутун) чунин муайян карда шудааст:

F1 = 1, F2 = 1, FK = FK - 2 + FK - 1, K = 3, 4,….

Ададҳои Фибоначии F1, F2,…, FN -ро чоп кунед.
